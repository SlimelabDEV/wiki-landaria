# 🌟 La Popularité !

La popularité est un système communautaire qui récompense les joueurs recommandés par d'autres.

Pour débuter, utilisez la commande :  
```/rc <joueur>```  
Cela ajoutera **+1 point de recommandation** au joueur ciblé.

---

:::tip ⏳ Limite de temps
Vous ne pouvez recommander qu'un joueur **toutes les heures**.
:::

Vous devez avoir joué au moins **5 heures** pour utiliser cette commande.  
Votre échelon est visible en passant la souris sur votre pseudo dans le chat.

---

## 🏅 Les échelons

| 🔢 Recommandations | Échelon                                       |
|--------------------|-----------------------------------------------|
| 0                  | Inconnu                                       |
| 10+                | 🔹 Nouveau                                    |
| 25+                | 🤝 Membre Actif                               |
| 50+                | 🧲 Apprécié                                   |
| 100+               | 💬 Estimé                                     |
| 250+               | 📇 On Le Connaît                              |
| 500+               | 👣 Habitué                                    |
| 750+               | 📘 Figure Publique                            |
| 1000+              | 🔥 Populaire                                  |
| 1500+              | 🎖 Respecté                                   |
| 2000+              | 🏆 Icône du Serveur                           |

---

## 🎁 Bonus

Chaque recommandation donne aussi :
- **100 XP de niveau**
- Une notification sonore
- Une annonce serveur si l'échelon monte

---

## 🛠 Commandes utiles

- `/rc <joueur>` : recommander un joueur
- `/echelons` : voir tous les échelons et leurs seuils
